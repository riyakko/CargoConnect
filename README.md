CargoConnect: Freight Forwarding & Logistics Tracker
Project Overview

CargoConnect is a web-based logistics management system designed to help small businesses efficiently book shipments, track cargo in real time, and manage freight operations. The system connects users with logistics providers, offering a streamlined platform for shipment booking, cost estimation, and delivery monitoring.

Purpose of the System
The goal of CargoConnect is to simplify the freight forwarding process by:

Providing an easy-to-use booking system
Offering real-time shipment tracking
Automating shipping cost calculations
Improving logistics transparency and efficiency

Key Features

Landing Page
Displays system overview and purpose
Navigation menu for easy access
Call-to-action buttons (Book Shipment, Track Cargo)
Preview of shipping rate calculator

Authentication System
User registration and login
Input validation (email and password)
Session management (login/logout)
Role-based access (Admin and User)

User Dashboard
View active, pending, and completed shipments
Display recent shipment activity
Quick action buttons (Book, Track, Generate Manifest)
Notifications and status updates

Booking System
Input shipment details (origin, destination, cargo type, size)
Select shipping method (container or truck)
Automatic cost estimation
Submit booking requests

Shipping Rate Calculator
Calculates cost based on distance, weight, and cargo type
Provides delivery time estimates
Real-time computation

Tracking System
Search using tracking ID
Displays shipment status (Pending, In Transit, Arrived)
Shows progress timeline
Real-time updates
Manifest Generator
Generates shipment documents
Preview, download, or print manifest

Admin Panel
Manage bookings and shipments
Approve or reject requests
Update shipment status
Manage user accounts
Monitor system activity

User Profile / Settings
Edit personal information
Change password
Manage account preferences
System Users
Customers – Book and track shipments
Admins – Manage operations and system data

Technologies Used
Frontend: HTML, CSS, JavaScript
Backend: PHP
Database: MySQL
Tools: Git, Visual Studio Code

How to Use
Register or log in to your account
Book a shipment by entering cargo details
View estimated cost and confirm booking
Track your shipment using the tracking ID
Download the shipment manifest if needed
